# Property Management System — Laravel

Sistema web de gestión de propiedades desarrollado como proyecto freelance en 2020.

## Descripción

Aplicación backend construida con Laravel para el control y administración de propiedades — gestión de registros, control de acceso por roles y organización de información mediante una arquitectura modular y mantenible.

## Características técnicas

- Arquitectura MVC con separación de responsabilidades — modelos, servicios y repositorios organizados por contexto
- Sistema de autenticación y control de acceso por roles
- Diseño de base de datos normalizado con MySQL
- Componentes de UI con Bootstrap integrados al backend Laravel
- Migraciones y seeders para control de versiones del esquema de base de datos

## Stack

- **Backend:** PHP, Laravel
- **Frontend:** Bootstrap, Blade Templates
- **Base de datos:** MySQL
- **Control de versiones:** Git

## Contexto

El repositorio refleja patrones de arquitectura Laravel aplicados en un entorno de producción real.
